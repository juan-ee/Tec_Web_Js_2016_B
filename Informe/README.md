## Enfasis de texto
* Para nosotros utilizar un texto en **negrilla** se utilizan dos "**"
* Un texto en *cursiva* simplemente utiliza un asterisco "*".
* Para tachar ~~un texto~~ se utiliza virbolillas "~~".
